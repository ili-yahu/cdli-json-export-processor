Help Information:

1. Use the 'Select JSON Files' button to select your files. Because the JSON files you get from the CDLI are not correctly formatted for SQL, they are automatically cleaned when selected.

2. Select or create a database to send the cleaned data to.

3. Click 'Send to SQLite' to export the cleaned data to the database. The process may take a few minutes depending on the size of the files.

For further assistance, please refer to the documentation on the GitHub repository: https://github.com/ili-yahu/database_manager.
